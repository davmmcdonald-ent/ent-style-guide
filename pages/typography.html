Sizing & Weight

Using REM (Root EM) units for font sizing is preferable over setting font sizes in PX (pixels) because REM units are relative to the root element (usually the HTML element). This ensures that font sizes scale proportionally based on the user's default font size settings in their browser. This adaptability enhances accessibility, as users with visual impairments often increase default font sizes to make text more readable. Additionally, REM units promote a more responsive design. When the base font size is adjusted, all elements sized with REM units will scale consistently, maintaining the design's visual hierarchy and proportions.

We use a font-size: 62.5% rule in the HTML CSS to make REM sizing calculations easier. Setting the base font size to 62.5% makes it easier to calculate REM values because 1rem will equal 10px. For instance, 1.6rem would be 16px, 2rem would be 20px, and so on. This simplification facilitates quick mental conversions and promotes a more intuitive approach to setting and adjusting font sizes across the design. By using a 62.5% base, designers can work with more straightforward numbers, reducing the likelihood of errors and ensuring a consistent, scalable typography system that adapts seamlessly to various screen sizes and user settings.

CSS

html { font-size: 62.5%; }
