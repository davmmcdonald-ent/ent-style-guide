Positioning Utilities

These utility classes help in managing the display and alignment of elements with minimal custom CSS. By using these predefined classes, you can ensure consistency and simplicity in your layout.

Alignment Styles

The .content--center class is used to center an element horizontally within its container.

CSS

.content--center { margin-left: auto; margin-right: auto; }

Display Utility Styles

The .display--block class is used to set an element's display property to block.

CSS

.display--block { display: block; }
