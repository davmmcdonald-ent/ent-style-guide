Flexbox Utilities

We prioritize using these core layout utility classes whenever possible to minimize CSS and leverage modularity. By utilizing these predefined flexbox classes, you can create flexible and responsive layouts with minimal custom CSS. Below, you'll find a categorized list of our flexbox utility classes.

Flexbox Container

The .flex class is used to apply flexbox display to a container.

CSS

.flex { display: flex; }

Flex Direction

These classes control the direction of the flex items within the container.

CSS

.flex--row { flex-direction: row; }

.flex--row-reverse { flex-direction: row-reverse; }

.flex--col { flex-direction: column; }

.flex--col-reverse { flex-direction: column-reverse; }

Justify Content

These classes align the flex items along the main axis within the flex container.

CSS

.flex--justify-start { justify-content: flex-start; }

.flex--justify-center { justify-content: center; }

.flex--justify-end { justify-content: flex-end; }

.flex--justify-between { justify-content: space-between; }

.flex--justify-around { justify-content: space-around; }

.flex--justify-evenly { justify-content: space-evenly; }

Align Items

These classes align the flex items along the cross axis within the flex container.

CSS

.flex--align-start { align-items: flex-start; }

.flex--align-center { align-items: center; }

.flex--align-end { align-items: flex-end; }

.flex--align-stretch { align-items: stretch; }

.flex--align-baseline { align-items: baseline; }

Gap Styles

These classes define the gap between flex or grid items. They can be used for both flexbox and CSS grid layouts.

CSS

.gap--16 { gap: 16px; }

.gap--24 { gap: 24px; }

.gap--32 { gap: 32px; }

.gap--64 { gap: 64px; }

Flex Item

The .flex__item class is used to make an element a flex item and define its flex property.

CSS

.flex__item { flex: 1; }
