Grid Utilities

We prioritize using these core layout utility classes whenever possible to minimize CSS and leverage modularity. By utilizing these predefined grid classes, you can create flexible and responsive layouts with minimal custom CSS. Below, you'll find a categorized list of our grid utility classes.

Grid Container

The .grid class is used to apply grid display to a container.

CSS

.grid { display: grid; }

Grid Columns

These classes define the number of columns in the grid layout.

CSS

.grid--2col { grid-template-columns: repeat(2, 1fr); }

.grid--3col { grid-template-columns: repeat(3, 1fr); }

.grid--4col { grid-template-columns: repeat(4, 1fr); }

Gap Styles

These classes define the gap between grid or flex items. They can be used for both CSS grid and flexbox layouts.

CSS

.gap--16 { gap: 16px; }

.gap--24 { gap: 24px; }

.gap--32 { gap: 32px; }

.gap--64 { gap: 64px; }
